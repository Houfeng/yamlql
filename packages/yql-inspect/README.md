# dn-template-react

这是一个 React 项目模板，默认安装了 react/react-dom，同时单元测试等基础设施都已配置完成。

## 使用

使用这个模板需要先安装好 Dawn [https://github.com/alibaba/dawn](https://github.com/alibaba/dawn)

### 初始化
```sh
mkdir demo
cd demo
dn init -t react
```

通过如上命令便可以完成项目的初始化

### 测试
```sh
dn test
```

### 开发
```sh
dn dev
```

### 构建
```sh
dn build
```

### 发布
```
dn publish
```

默认的发布配置是「发布到 npm」，请根据需要更改 `.dawn/pipe.yml` 进行 `publish` 配置
